# 02 — Statistics & Math for AI

The mathematical backbone of every ML algorithm.

## Contents

| Folder | Topics |
|--------|--------|
| `01_descriptive_stats/` | Mean, median, variance, distributions, visualisation |
| `02_probability/` | Bayes theorem, conditional probability, common distributions |
| `03_inferential_stats/` | Hypothesis testing, p-values, confidence intervals, A/B testing |
| `04_linear_algebra/` | Vectors, matrices, eigenvalues, SVD, PCA |
| `05_calculus_optimization/` | Derivatives, gradients, chain rule, gradient descent variants |
