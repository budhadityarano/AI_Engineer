# 13 — Coding Problems (DSA + System Design)

Most-asked data structures, algorithms, and system design problems for AI/SWE interviews.

## Contents

| Folder | Topics |
|--------|--------|
| `01_arrays_strings/` | Two pointers, sliding window, hashing, prefix sums |
| `02_trees_graphs/` | BFS/DFS, binary trees, tries, union-find, shortest path |
| `03_dynamic_programming/` | 1D/2D DP, knapsack, LCS, LIS, memoisation patterns |
| `04_sorting_searching/` | QuickSort, MergeSort, binary search variants, heap |
| `05_system_design/` | ML system design: recommendation, search, fraud detection, LLM serving |
