# 06 — Deep Learning

From perceptrons to Transformers — implemented from scratch and with frameworks.

## Contents

| Folder | Topics |
|--------|--------|
| `01_neural_networks/` | Perceptron, backprop from scratch, activation functions, optimisers |
| `02_cnn/` | Conv layers, pooling, ResNet, transfer learning, image classification |
| `03_rnn_lstm/` | Sequence models, vanishing gradients, LSTM/GRU, sequence-to-sequence |
| `04_transformers/` | Attention mechanism, BERT, GPT architecture, positional encoding |
| `05_frameworks/` | PyTorch & TensorFlow/Keras side-by-side implementations |
